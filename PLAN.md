# План реализации Marketing Platform

## Фаза 0: Инфраструктура (1 шаг)
1. **Включить middleware авторизации** — раскомментировать `updateSession()` в middleware.ts
2. **Проверить что schema.sql накачен** — проверить наличие таблиц в Supabase
3. **Создать seed-данные** — заполнить БД тестовыми проектами, задачами, постами чтобы было с чем работать
4. **Создать слой запросов** — `src/lib/supabase/queries.ts` — все Supabase-запросы в одном месте (проекты, задачи, посты, бюджет, аналитика)

## Фаза 1: Проекты
5. **Список проектов** — заменить hardcoded demoProjects на запрос из Supabase, фильтрация по типу
6. **Страница проекта [id]** — динамическая загрузка по ID из URL, вкладки: обзор, команда, бюджет
7. **CRUD проектов** — модальное окно создания/редактирования проекта, удаление
8. **Управление командой** — добавление/удаление участников проекта

## Фаза 2: Задачи (Kanban)
9. **Kanban-доска из БД** — загрузка колонок и задач из task_boards/task_columns/tasks
10. **Создание задач** — модальное окно: название, описание, приоритет, категория, исполнитель, дедлайн
11. **Drag & drop** — перетаскивание задач между колонками (react-beautiful-dnd или @dnd-kit)
12. **Фильтры и "Мои задачи"** — фильтрация по проекту, приоритету, категории; вкладка "Мои задачи"

## Фаза 3: Контент-план
13. **Контент из БД** — загрузка постов из content_posts, отображение в календаре и списке
14. **Создание/редактирование постов** — модальное окно с полями: платформа, тип, дата, текст, медиа
15. **Статусы и согласование** — workflow: draft → in_review → approved → scheduled → published

## Фаза 4: Dashboard
16. **Реальные KPI** — агрегированные запросы: расходы, задачи, посты, бюджет из БД
17. **Активность и дедлайны** — activity_log + задачи с ближайшими дедлайнами
18. **Динамический userName** — имя текущего пользователя в sidebar из profiles

## Фаза 5: Аналитика и Ads API
19. **Установить Recharts** — графики расходов по дням
20. **Meta Ads API** — подключение аккаунта, синхронизация кампаний и метрик в ad_campaigns/ad_metrics_daily
21. **Google Ads API** — аналогично Meta Ads
22. **Вкладки аналитики** — Overview, Meta Ads, Google Ads, Сравнение

## Фаза 6: Настройки и уведомления
23. **Профиль пользователя** — редактирование имени, телефона, аватара
24. **Управление пользователями** — список пользователей, назначение ролей (admin/manager/marketer/viewer)
25. **Уведомления** — задача назначена, дедлайн, контент на согласовании
26. **Интеграции** — страница подключения рекламных кабинетов Meta/Google
